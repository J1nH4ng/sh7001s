# utils-shell

> [!NOTE]
> 工程化构建脚本工具集

## 目录结构

```bash
/
  core/  ：核心脚本文件，不依赖于任何其他脚本文件
  share/ ：公共共享部分，仅依赖 core 目录下文件
  utils/ ：工具脚本文件
```

## 主要的功能

core/ 目录下：
    - Banner 输出功能
    - 脚本执行的日志输出功能

share/ 目录下：
    - 文件的 md5 和 sha256 生成和校验功能
    - 文件同步功能

utils/ 目录下：

    - log 文件中关键信息抓取功能
